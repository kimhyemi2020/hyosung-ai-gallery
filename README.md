인천효성고 시민개발자 갤러리 v3

1. 기존 Apps Script는 그대로 사용해도 됩니다.
2. Apps Script 웹 앱 URL을 복사합니다.
3. index.html 파일을 열고 아래 줄을 찾습니다.

const API_URL = "여기에_Apps_Script_웹앱_URL을_붙여넣으세요";

4. 따옴표 안에 웹 앱 URL을 넣습니다.
5. 저장 후 GitHub Pages 또는 Netlify에 업로드합니다.

기능:
- 승인=1인 작품만 표시
- 최신순 / 인기순 / 작품명순
- 좋아요
- 댓글
- 검색
- 전체 작품 수, 좋아요 수, 댓글 수 표시
- 모바일 화면 지원
- 같은 기기에서 좋아요 중복 방지
